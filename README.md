# vue-agency

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Install server simulated a production

```
npm install -g serve
```

### Run the new serve

```
serve -s dist
```
